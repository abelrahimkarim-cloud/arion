# Streetwear E-Commerce Full Stack App

Modern multi-product streetwear store with a Next.js 15 storefront and Laravel 12 API backend.

## Tech Stack

- Frontend: Next.js 15, React, TypeScript, Tailwind CSS
- Backend: Laravel 12 REST API
- Database: MySQL
- Authentication: JWT (Admin only)
- COD-only checkout flow
- Responsive design for Mobile, Tablet, Desktop

## Project Structure

- `frontend/` – Next.js storefront and admin UI
- `backend/` – Laravel REST API, migrations, seeders
- `docs/` – Documentation, deployment guide, schema, API spec
